Hex and RGB are meant to be read by machines, making it difficult for humans to understand. HSL is the opposite and is made for human comprehension so it is easily comprehensible.

Preferred color code format: Hex

HWB summary: HWB stands for hue, whiteness, blackness. It uses percentages of each to make out the color.

CMYK summary: CMYK stands for cyan, magenta, yellow, and black. It used these for colors to combine to make a color like RGB does. This is because printers usually rely on CMYK while computer screens use RGB.
